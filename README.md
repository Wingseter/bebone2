# bebone2
코로나 명부를 작성하는 프로그램 bebone의 두번째 버전

## bebone이란?
코로나가 지속됨에 따라 뼈다귀가 되는것 같은 것을 표현한다. ☠️

## 목적
코로나 명부를 작성하는 프로그램으로 
명부 추가가 가능하다

id, 이름, 날짜를 통한 검색이 가능하며
검색된 목록을 프린터로 출력이 가능하다.
